Ejemplo 4: ng-repeat y ng-if | show

Lenguajes de programación

Mostrar lista Ocultar lista
  • {{$index}}) Lenguaje -> {{lang.name}} :: [{{lang.layer}}]

Descripción

Mostrando colecciones

Para imprimir una colección (array) utilizaremos una construcción "out-of-the-box" o built-in, la directiva ng-repeat, cuyo funcionamiento es similar a for-each.

En el controlador expondremos una función (add) mediante la cual añadiremos un nuevo elemento a dicho array, tomando como entradas los dos input (nombre y capa).

Por último, se ha añadido también un flag que controlará la visibilidad del nodo del DOM sobre el que está aplicado; en el ejemplo, mostramos u ocultamos toda la lista.

              
                var app = angular.module('ExampleApp4', []);
                app
                  .controller('ctrl1', function($scope) { 
                    $scope.isLanguageListVisible = true;
                    $scope.languages = [
                      { name: 'Java', layer : 'BE' },
                      { name: 'Go', layer : 'BE' },
                      { name: 'JavaScript', layer : 'FE' },
                      { name: 'Scala', layer : 'BE' },
                      { name: 'Python', layer : 'BE' },
                      { name: 'Clojure', layer : 'BE' },
                      { name: 'Ruby', layer : 'BE' }
                    ];

                    $scope.add = function(n, l) {
                      $scope.languages.push({name:n, layer:l});
                      $scope.n = '';
                      $scope.l = '';
                    }
                  });

En la plantilla (vista), insertamos: 

              
                <div ng-controller="ctrl1">
                  <h1>Lenguajes de programación</h1>
                  <input type="checkbox" ng-model="isLanguageListVisible" />
                  <span ng-if="!isLanguageListVisible">Mostrar lista</span>
                  <span ng-if="isLanguageListVisible">Ocultar lista</span>
                  <ul ng-show="isLanguageListVisible">
                    <li ng-repeat="lang in languages">
                      { {$index} }) Lenguaje -> { {lang.name} } :: [{ {lang.layer} }]
                    </li>
                  </ul>
                  <label>Nuevo lenguaje: </label><input ng-model="n" /><input ng-model="l" />
                  <button ng-click="add(n, l)">Añadir lenguaje</button>
                </div>
