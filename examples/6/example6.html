Ejemplo 6: Filtro + directiva

Lenguajes de programación

  • {{$index}}) Lenguaje -> {{lang.name}} :: [{{lang.layer}}]

Descripción

Cuando utilizamos la directiva ng-repeat podemos aplicar un filtro sobre la tabla a modo de filtro.

Dicho filtro será alimentado por un campo de texto; este texto será el criterio de búsqueda sobre los campos de la tabla (que podrá ser sobre una o todas las columnas; en el ejemplo, toda la tabla).

              
                var app = angular.module('ExampleApp1', []);
                app
                  .controller('ctrl1', function($scope) { 
                    $scope.languages = [
                      { name: 'Java', layer : 'BE' },
                      { name: 'Go', layer : 'BE' },
                      { name: 'JavaScript', layer : 'FE' },
                      { name: 'Scala', layer : 'BE' },
                      { name: 'Python', layer : 'BE' },
                      { name: 'Clojure', layer : 'BE' },
                      { name: 'Ruby', layer : 'BE' }
                    ];
                  });

En la plantilla (vista), insertamos: 

              
                <div ng-controller="ctrl1">
                  <h1>Lenguajes de programación</h1>
                  <label>Filtro: </label><input ng-model="filterCriteria" />
                  <ul>
                    <li ng-repeat="lang in languages | filter:filterCriteria">
                      { {$index} }) Lenguaje -> { {lang.name} } :: [{ {lang.layer} }]
                    </li>
                  </ul>
                </div>
