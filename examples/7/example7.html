Ejemplo 7: Servicios

Calculador CodeMotion 2015

Resultado: {{result}}

Suma automática: {{number1 + number2}}

Descripción

Se define un servicio para encapsular la lógica común o específica; en este caso el servicio simula dos operaciones básicas de una calculadora: suma y multiplicación.

De las dos funcionalidades implementadas (expuestas a través de una factoría), el resultado del cálculo se obtendrá.

  • Resultado: se invoca al servicio cuando se pulsa sobre el botón.
  • Suma automática: aprovechando la característica del two-way data-binding, la operación de multiplicación se realiza y actualiza de manera automática, sin necesidad de clicar ningún botón manualmente. 

              
                var app = angular.module('ExampleApp1', []);
                app
                  .controller('ctrl1', ['$scope', 'CalcService', function($scope, calculator) { 
                    $scope.sum = function() {
                      $scope.result = calculator.add($scope.number1, $scope.number2);
                    }

                    $scope.mult = function(x, y) {
                      $scope.result = calculator.multiply(x, y);
                    }
                  }])
                  .factory('CalcService', function() {
                    return {
                      add : function(a, b) { return a + b; },
                      multiply : function(a, b) { return a * b; }
                    }
                  });

En la plantilla (vista), insertamos: 

              
                <div ng-controller="ctrl1">
                  <h1>Calculador CodeMotion 2015</h1>
                  <label>Operando 1:</label><input type="number" ng-model="number1" />
                  <label>Operando 2:</label><input type="number" ng-model="number2" />
                  <h3>Resultado: { {result} }</h3>
                  <h3>Suma automática: { {number1 + number2} }</h3>
                  <button ng-click="sum()">Suma</button> 
                  <button ng-click="mult(number1, number2)">Multiplica</button>
                </div>
